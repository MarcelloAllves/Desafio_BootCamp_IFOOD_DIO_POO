Ao criar o projeto necessitamos sincroniza-lo com um repositório criado no github.
No Intellij na barra de ferramenta localizado na parte superior da IDE, localizar VCS.
Em VCS selecionar a opção "Enable Version Control Integration".
Na tela que abrir selecionar a opção Git e apertar OK.
Nota que onde estava escrito VCS mudou para Git.
Na aba projeto localizar a pasta do projeto, abrir a arvore da pasta .idea, nela vai ter um arquivo com o nome de .gitignore.
No endereço https://www.toptal.com/developers/gitignore digitar Java Intellij depois apertar o botão criar.
Na página que abrir copiar todo o conteúdo e colar logo abaixo do ultima linha no documento do gitignore.
